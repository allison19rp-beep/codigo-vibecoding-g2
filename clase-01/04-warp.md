Warp es una terminal moderna con autocompletado inteligente y una interfaz mucho más amigable que la terminal clásica. **Si no te funciona, no hay problema** — hay varias alternativas excelentes.

---

## Instalar Warp

1. Ve a [https://warp.dev](https://warp.dev)
2. Descarga para tu sistema operativo
3. Instala y abre
4. Te pedirá crear una cuenta gratuita

---

## ¿No te funcionó Warp? Alternativas

### Para Windows

| Herramienta | Por qué usarla | Cómo obtenerla |
|---|---|---|
| **Git Bash** | Terminal estilo Linux para Windows, la más recomendada del curso | Instala Git desde [git-scm.com](https://git-scm.com) — viene incluida |
| **Windows Terminal** | Terminal moderna y limpia de Microsoft | Búscala en la Microsoft Store |
| **PowerShell 7** | Versión moderna y mejorada de PowerShell | [github.com/PowerShell/PowerShell](https://github.com/PowerShell/PowerShell) |
| **Terminal de Cursor** | La más fácil — ya está instalada si tienes Cursor | `` Ctrl + ` `` dentro de Cursor |

**Recomendación para Windows si Warp no funciona:** usa la **terminal de Cursor** o instala **Git Bash**. Son las más simples para empezar.

---

### Para Mac

| Herramienta | Por qué usarla | Cómo obtenerla |
|---|---|---|
| **Terminal** | La que ya viene con Mac, funciona perfectamente | Ya está en tu Mac (`Cmd + Espacio` → Terminal) |
| **iTerm2** | Alternativa popular con más funciones | [iterm2.com](https://iterm2.com) |
| **Terminal de Cursor** | La más fácil — ya está instalada si tienes Cursor | `` Ctrl + ` `` dentro de Cursor |

**Recomendación para Mac si Warp no funciona:** la **Terminal que ya tienes en Mac** es más que suficiente para todo el curso.

---

## ¿Cuál elegir?

