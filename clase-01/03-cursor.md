Cursor es el editor de código que usaremos durante el curso. Es similar a VS Code pero con inteligencia artificial integrada que te ayuda a escribir código más rápido.

---

## Instalación

1. Ve a [https://cursor.sh](https://cursor.sh)
2. Haz clic en el botón de descarga para tu sistema operativo
3. Instala como cualquier programa
4. Abre Cursor

---

## Primera vez que lo abres

Al abrir Cursor por primera vez:

- Te preguntará si quieres importar tu configuración de VS Code — puedes hacerlo si ya lo tenías instalado
- Te pedirá crear una cuenta gratuita en cursor.sh
- El plan gratuito incluye un número limitado de usos de IA por mes, suficiente para empezar

---

## Terminal integrada

Cursor tiene una terminal incorporada. Para abrirla:

| Sistema | Atajo |
|---|---|
| **Mac** | `` Ctrl + ` `` (la tecla del acento grave) |
| **Windows** | `` Ctrl + ` `` |

O también desde el menú: `View → Terminal`

Esta terminal funciona igual que la terminal de tu sistema. Puedes usarla para correr Claude Code u OpenCode directamente desde el editor.

---

## Atajos básicos útiles

| Acción | Mac | Windows |
|---|---|---|
| Abrir/cerrar terminal | `` Ctrl + ` `` | `` Ctrl + ` `` |
| Abrir un archivo | `Cmd + P` | `Ctrl + P` |
| Guardar archivo | `Cmd + S` | `Ctrl + S` |
| Buscar en el archivo | `Cmd + F` | `Ctrl + F` |
| Buscar en todo el proyecto | `Cmd + Shift + F` | `Ctrl + Shift + F` |
| Abrir carpeta | `Cmd + O` | `Ctrl + O` |
